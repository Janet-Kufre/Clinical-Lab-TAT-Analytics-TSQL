# Clinical-Lab-TAT-Analytics-TSQL

## Project Overview
This project simulates a hospital laboratory system used to track patient tests and analyze laboratory turnaround time (TAT).

## Objectives
- Design relational database for patient and lab data
- Perform data cleaning and updates
- Analyze test performance and delays
- Identify high-risk patients with critical results

## Tools Used
- SQL (DDL, DML, Joins, Aggregations)
- MySQL / SQL Server compatible syntax

## Key Insights
- Turnaround time varies across test types
- Critical results require immediate attention
- Some tests exceed SLA thresholds
